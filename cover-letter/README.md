# Cover Letter Template — Usage Notes

This directory contains a **generic cover letter template** intended to accompany the CV stored in this repository.

## Purpose

The template is designed to:

- Be suitable for **senior backend / principal-level roles**
- Remain **role-agnostic** and reusable
- Reflect a professional, restrained tone appropriate for:
  - regulated environments
  - backend-heavy positions
  - technically mature teams

It is intentionally not customised for any specific company or role.

## How to use the template

When applying for a specific position:

1. Make a copy of `template.md` outside this repository.
2. Apply **minimal customisation only**, typically:
   - Adjusting one paragraph to reflect the role or domain
   - Optionally mentioning the problem space or system scale
3. Keep the letter to **one page**.

Do not:

- Add company names inline unless required
- Add emotional or promotional language
- Rewrite the entire letter per application

Consistency and restraint are intentional.

## What this repository does not contain

- Application-specific cover letters
- Recruiter correspondence
- Salary, availability, or visa details

Those are deliberately kept separate.
